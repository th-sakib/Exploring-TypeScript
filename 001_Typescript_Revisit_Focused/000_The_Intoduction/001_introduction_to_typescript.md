# Introduction to Typescript
## What is Typescript?
- Typescript is a statically typed superset of Javascript which compiles to JS.
- Typescript gives better tooling support like autocompletion, refactoring and dubugging.

## Benifits of typescript?
- Typescript gives us ability to detect error in runtime.
- It provide type safety by static typing.
- It gives better tooling support.
- It increases code readablity.
- It is better for team collaboratative projects.

## Installing **NodeJS**
1. install `nvm` or `fnm` as package managers. Which manages `nodejs` packages.
2. use the package manager to install and manage `node`.

## Installing **Typescript**
1. Install typescript from npm repository `npm install typescript`.
2. use `tsc` command to interact with `.ts` files.

## Some basic commands
1. `tsc -v` - check version. 
2. `tsc <typescriptFile>` - Compile typescript file into Javascript.
3. `tsc --init` - create typescript configuration file `./tsconfig.json`.
